> Backend

1. Instalar los paquetes de node

1.1 Ir a la carpeta del frontend

``` cd frontend-meli ```

1.2 Luego

``` npm run install ```

2. Configurar el puerto

El puerto por defecto será el 4000, si no debo se deberá especifar en el .env del backend-meli

Asi

```PORT=4000```

3. Lanzar la app

``` npm run dev ```
